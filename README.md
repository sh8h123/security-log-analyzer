# Security Log Analyzer

A Python security tool that scans logs for suspicious activity.

## Features

- Counts failed login attempts
- Detects brute-force attack indicators
- Flags security alerts
- Generates a security summary

## Example Output

=== SECURITY LOG ANALYZER ===

[+] Total Lines Scanned: 6
[!] Suspected Brute-Force Attempts Found: 3
[CRITICAL] Security Alerts Flagged: 1

--> 2026-06-23 12:03:40 ALERT Brute-force flag triggered for IP 192.168.1.50
